# Models

This folder contains the curated detector used by the submitted demo.

```text
bmd45_yolo11s_gpu_50epochs_best.pt
```

The weight file was produced by a YOLO11s adaptation run on a 1000-image BMD-45 training subset and evaluated on a 200-image validation/demo subset using an NVIDIA GeForce RTX 4060 Laptop GPU.

See `outputs/training_summary.md` for the exact command, metrics, and RT-DETR-L comparison.
